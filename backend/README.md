# How to run

- Create Virtual Environment
  
    ```sh
    python3 -m venv .venv
    ```

- Activate venv
  
    ```sh
    source .venv/bin/activate
    ```

- Install Requirements

    ```sh
    pip install -r requirements.txt
    ```

- Run Backend Server

    ```sh
    python3 manage.py runserver
    ```

- Note:- Admin username `root` and password `root`
